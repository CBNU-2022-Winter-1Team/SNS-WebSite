# SNS-WebSite

특색 있는 익명성을 보장하는 sns 웹사이트
<br>

## 사용 가이드
npm install : package.json 파일의 의존성을 설치하는 동작    
npm start : react 실행 시키기



## 기능
 - 로그인    
 - 회원가입    
 - 게시판 작성, 수정, 삭제    
 - 비밀글 설정    
 - 스크랩    
 - 익명 설정    
 - 팔로우    
 - 채팅    
 - 알림    
 - 신고
